# Build Here or Migrate — deck plan

8 slides, 1920×1080, educational audience, English.
Aesthetic: ornate Baroque/Rococo. Gold accents, ornamental rules, jewel tones + soft ivory.

## Title sequence (short noun-phrase style)
1. Build Here or Migrate
2. The Crossroads
3. A Country on the Move
4. The Case for Migrating
5. The Case for Building Here
6. The Third Path: Remote Work
7. The Hidden Price
8. What Would You Choose?

Reads as a clean ToC: question → dilemma → data → pro-leave → pro-stay → hybrid → costs → conclusion. ✓

## Design system
- Fonts: Cormorant Garamond (display), Pinyon Script (script eyebrows), EB Garamond (body)
- Backgrounds (2): deep emerald `--bg-dark` and warm ivory `--bg-light`
- Gold: #c9a14d range; gilded frames = layered borders
- Ornaments: gold double rules with ◆ center, fleurons ❧, Roman numerals for list points
- Case slides (4, 5, 6) share one layout: framed painting column + 3 Roman-numeral points
